# ModernTech HR Management System

# 📖 Overview

The **ModernTech HR Management System** is a fully responsive Human Resource Management (HRM) web application designed to simplify the management of employees, payroll, attendance, reviews, and administrative tasks within an organisation.

The application provides a modern dashboard where administrators can access employee information, manage payroll, monitor attendance, schedule company events, track performance reviews, and customize the application's appearance through multiple colour themes and dark mode.

The project was developed using **HTML5**, **CSS3**, and **Vanilla JavaScript**, with JSON files serving as the project's data source and Local Storage used to store user-specific information.

---

# 🎯 Project Objectives

The purpose of this project is to:

- Digitize employee records.
- Improve payroll management.
- Simplify attendance tracking.
- Provide a centralized HR dashboard.
- Improve the user experience through responsive design.
- Demonstrate front-end web development skills using JavaScript, HTML, and CSS.
